# Expense Tracker

## Overview

The Expense Tracker is a full-stack web application designed to help users manage their finances by tracking income and expenses. Users can categorize their transactions, visualize their spending habits, and generate reports. The app utilizes ReactJS for the frontend, Node.js for the backend, MongoDB for data storage, and JWT for secure authentication.

## Features

- User authentication using JWT
- Add, edit, and delete income and expense transactions
- Categorize transactions for better organization
- Visualize spending habits with charts and graphs
- Generate reports to analyze financial performance
- User-friendly interface with responsive design

## Technologies Used

- **Frontend:** ReactJS
- **Backend:** Node.js with Express
- **Database:** MongoDB
- **Authentication:** JSON Web Tokens (JWT)
